Maven enhancement plugin

If you do not want to use maven tiles you can specify the maven enhancement plugin in 'normal' fashion. 


  io.ebean
  ebean-maven-plugin
  14.1.0
  
    
    
      main
      process-classes
      
        debug=1
      
      
        enhance
      
    

    
    
      test
      process-test-classes
      
        debug=1
      
      
        testEnhance
