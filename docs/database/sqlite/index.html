Sqlite

Testing

To test using Sqlite set the platform to sqlite in src/test/resources/application-test.yaml

Refer to docs / testing if application-test.yaml doesn't exist yet.

```yml ebean: test: platform: sqlite # h2, postgres, mysql, oracle, sqlserver ddlMode: dropCreate # none | dropCreate | migrations dbName: test ```

The above will use the following defaults:

url:jdbc:sqlite:{databaseName}
driver:org.sqlite.JDBC
isolationlevel:read_uncommitted
